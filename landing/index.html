Installation Guide

Easy 4-Step Setup

No technical knowledge required. Follow the steps below and you'll be ready in under 2 minutes.

1

Download the Extension

Get the source code from GitHub

Go to the GitHub repository. Click the green Code button → Download ZIP. Then unzip the downloaded file to a folder you can easily find (e.g., your Desktop).
2

Open Chrome Extensions

Navigate to the extensions manager

Open Google Chrome. In the address bar, type chrome://extensions and press Enter. You'll see a page showing all your installed extensions.
3

Enable Developer Mode

A one-time toggle to allow loading extensions

In the top-right corner of the Extensions page, find the Developer mode toggle and turn it ON. This is a standard Chrome option for installing extensions that aren't in the Chrome Web Store.
4

Load the Extension

Point Chrome to your downloaded folder

Click the Load unpacked button that now appears. In the file picker, navigate to and select the unzipped folder from Step 1. Click Select Folder. The AI Translator icon will appear in your Chrome toolbar!

Last Step

Set Up Your Free API Key

The extension uses Google Gemini AI. Getting a free API key takes only 1 minute.

1

Visit Google AI Studio

Go to aistudio.google.com/apikey and sign in with your Google account.

2

Click "Create API Key"

Click the blue "Create API key" button. Select "Create API key in new project". Your key will be generated instantly.

3

Copy the Key

Click the copy icon next to your API key. It looks like a long string of random letters and numbers starting with "AIza...".

4

Paste into Extension Settings

Click the AI Translator icon in your Chrome toolbar → Open Settings (or press Alt+S) → Paste your API key → Save. You're all set!

The free Gemini API quota is generous for personal use. Your API key is stored only on your device (Chrome local storage) and is never sent anywhere except to Google's servers to process your requests.
